---
title: pulse workspace
description: Create, list, inspect, and abandon workspaces.
---

```bash
pulse workspace <create|list|status|abandon>
```

## workspace create

```bash
pulse workspace create --intent <text> [--scope <glob>...]
```

Create a new workspace branching from the current main head.

| Flag | Description |
|------|-------------|
| `--intent <text>` | What this workspace is for (required) |
| `--scope <glob>` | File patterns this workspace touches (repeatable) |

Pulse checks for scope overlaps with other active workspaces and reports them.

## workspace list

```bash
pulse workspace list [--all]
```

List workspaces. By default, only active workspaces are shown.

| Flag | Description |
|------|-------------|
| `--all` | Include merged and abandoned workspaces |

## workspace status

```bash
pulse workspace status <id>
```

Show details for a specific workspace: status, intent, scope, author, base changeset, and number of commits.

## workspace abandon

```bash
pulse workspace abandon <id>
```

Mark a workspace as abandoned. Its commits remain in storage but are excluded from active listings.
