---
title: pulse merge
description: Merge a workspace into trunk.
---

```bash
pulse merge <workspace>
```

Close a workspace by merging its changes into trunk.

## Merge strategies

**Fast-forward** — if trunk hasn't moved since the workspace was created, the workspace's latest snapshot becomes the new trunk head directly.

**Three-way merge** — if trunk has advanced, Pulse compares the ancestor snapshot (workspace base), trunk snapshot, and workspace snapshot. Non-conflicting changes from both sides are combined.

**Conflict** — if both sides modified the same file, Pulse reports the conflict. Trunk is not modified and the workspace stays active.

## Example: success

```bash
pulse merge ws-c348
```

```
Merged into trunk. Changeset: 68fe576d
  files: src/auth.rs, src/middleware.rs
```

## Example: conflict

```bash
pulse merge ws-a1b2
```

```
Merge conflict!
  conflict: src/shared.rs
  trunk snapshot:     a1b2c3d4
  workspace snapshot: e5f6a7b8
```

The workspace remains active so you can resolve the conflict and try again.
